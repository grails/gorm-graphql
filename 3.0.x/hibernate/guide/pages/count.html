Version: 3.0.0

3.4 Count

In this example the query is demonstrating retrieving the total count of speakers.

curl -X "POST" "http://localhost:8080/graphql" \
     -H "Content-Type: application/graphql" \
     -d $'
{
  speakerCount
}'

which returns:

{
    "data": {
        "speakerCount": 7
    }
}
